# Zap-GPT

本项目探索将 ChatGPT 与 WhatsApp 集成，将聊天机器人转变为虚拟助手，能够执行与朋友聊天、回答客户问题等任务，并在对话中增添人性化的触感。

## 📚 工作原理

集成从 [wppconnect](https://github.com/wppconnect-team/wppconnect) 开始，它负责建立与 WhatsApp 的连接。<br/>
收到的消息随后由 ChatGPT 或 Gemini API 处理，生成连贯且个性化的回复。<br/>
我们使用 OpenAI 的 [Assistant](https://platform.openai.com/docs/assistants/overview)，这是一个预配置了详细提示词的 OpenAI 模型。<br/>
对于 Gemini，我们使用预设的提示词来指导模型。<br/>
这些提示词指导助手如何以连贯和个性化的方式回复，确保互动不仅保持相关性和吸引力，还能体现自然、人性化的对话方式。
